## What is BodyWorks?

BodyWorks is built with modern web technologies including Next.js 15, React 19, TypeScript, React Query, and Shadcn UI. The platform connects to the Body Works API to deliver a seamless fitness experience with powerful search and filtering capabilities.

<Note>
  BodyWorks is completely free to use. Access the live platform at [bodyworks.akshatjaiswal.me](https://bodyworks.akshatjaiswal.me)
</Note>

## Key features

BodyWorks offers a comprehensive suite of features designed to support your fitness journey:

### Massive exercise database

Browse over **1300+ exercises** with detailed instructions, animations, and muscle targeting information. Each exercise includes:

- High-quality GIF demonstrations
- Step-by-step instructions
- Target muscle information
- Equipment requirements
- Body part focus

### Advanced filtering system

Find exactly what you need with powerful filtering options:

- **30+ Equipment types** - Filter by barbell, dumbbell, resistance bands, body weight, and more
- **20+ Target muscles** - Focus on specific muscle groups like biceps, quadriceps, or deltoids
- **10+ Body parts** - Filter exercises by back, chest, legs, arms, and other body regions

### Structured workout routines

Access over **600+ pre-built workout routines** ranging from 1 week to 52 weeks:

- Beginner-friendly programs
- Intermediate and advanced routines
- Gender-specific workouts
- Goal-oriented plans (strength, hypertrophy, endurance)

### Responsive design

BodyWorks is fully responsive and works seamlessly across all devices - desktop, tablet, and mobile. Train anywhere, anytime.

### Modern tech stack

Built with cutting-edge technologies for optimal performance:

- **Next.js 15** with Turbopack for lightning-fast builds
- **React 19** for modern component architecture
- **TypeScript** for type safety
- **React Query** for efficient data fetching and caching
- **Shadcn UI** for beautiful, accessible components
- **Tailwind CSS 4** for utility-first styling
- **Motion (Framer Motion)** for smooth animations
